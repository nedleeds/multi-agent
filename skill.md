# fzf 사용법

## 개요

`fzf`는 명령 줄에서 사용할 수 있는 상호작용형 필터 프로그램으로, 리스트의 항목들을 "퍼지" 매칭 알고리즘을 통해 필터링합니다. 사용자는 일치하지 않는 문자를 생략하고도 원하는 결과를 빠르게 찾을 수 있습니다.

## 사용법

명령어:
```
fzf [options]
```

### 주요 옵션

- `-x, --extended`: 확장 검색 모드. 기본적으로 활성화되어 있으며, `+x` 또는 `--no-extended`로 비활성화할 수 있습니다.
- `-e, --exact`: 정확한 일치 검색을 활성화합니다.
- `-i, --ignore-case`: 대소문자를 구분하지 않는 검색을 수행합니다. (기본값: 스마트 케이스 매치)
- `+i, --no-ignore-case`: 대소문자를 구분하는 검색을 수행합니다.
- `--literal`: 검색을 위한 라틴 문자 스크립트를 정규화하지 않습니다.
- `--scheme=SCHEME`: 입력 유형에 맞춘 스코어링 스킴을 선택합니다. 기본값은 `default`이며, 경로에 맞춘 스킴은 `path`입니다.

## 예제

- `ls | fzf`: 현재 디렉토리의 파일 목록을 퍼지 검색으로 필터링합니다.
- `git branch | fzf`: Git 브랜치 목록을 퍼지 검색으로 필터링합니다.

더 많은 옵션과 사용법은 `man fzf`를 참조하십시오.
